# لوحة إدارة المشتركين - React/TypeScript

واجهة عربية مبنية بـ React و TypeScript لإدارة مدن العملاء، إضافة المشتركين، وتتبع حالة الاشتراك والسداد شهرياً مع تخزين البيانات في المتصفح.

## الأوامر الأساسية
- `npm install` لتثبيت الاعتمادات.
- `npm run dev` لتشغيل نسخة التطوير.
- `npm run build` لبناء الإصدار الجاهز للنشر.
- `npm run preview` لمعاينة البناء المنتج.
- `npm run lint` للتحقق من صحة إعدادات TypeScript.
